¿Cómo funciona?

1

Obtención de ubicación

El módulo NEO-8M recibe señales satelitales y obtiene coordenadas GPS.

2

Envío de datos

El ESP32 se conecta a la zona WiFi del celular del usuario para enviar la ubicación.

3

Consulta familiar

Los familiares ingresan el código del bastón y visualizan la última ubicación registrada.
