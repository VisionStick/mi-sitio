Vision Stick:
un bastón inteligente para apoyar la movilidad segura.

Vision Stick busca ayudar a personas con discapacidad visual mediante sensores, alertas por vibración y una estructura práctica que facilite detectar obstáculos durante el desplazamiento.

Sobre el proyecto

¿De qué trata Vision Stick?

Es una propuesta tecnológica que integra componentes electrónicos en un bastón para entregar alertas simples, rápidas y útiles al usuario.

Problema

Los obstáculos inesperados pueden dificultar la movilidad y aumentar el riesgo de accidentes en personas con discapacidad visual.

Solución

Crear un bastón con sensores ultrasónicos, vibradores y una placa ESP32 que detecte obstáculos y avise a la persona.

Objetivo

Desarrollar un prototipo funcional, económico y fácil de explicar, documentando cada avance del proceso.

Materiales utilizados

Cada material cumple una función dentro del prototipo. Haz clic en cada tarjeta para ver su utilidad.

Funciona como el cerebro del proyecto. Recibe la información de los sensores, procesa las distancias y activa las alertas del bastón.

Detectan obstáculos cercanos midiendo distancia. Permiten reconocer elementos al frente o en zonas específicas del bastón.

Entregan alertas físicas mediante vibración. Pueden indicar cercanía, dirección o intensidad del obstáculo detectado.

Sirve para monitorear el prototipo durante pruebas, mostrando datos como distancia, estado de sensores o funcionamiento general.

Alimenta el sistema electrónico y permite recargar el prototipo sin depender siempre de una conexión directa al computador.

Permiten realizar las conexiones entre componentes. La protoboard ayuda en pruebas y la PCB puede usarse para una versión más ordenada.

Forma la base física del bastón. Debe ser liviano, resistente y cómodo para integrar sensores, batería y cables.

Suministrara la energia de la bateria, controlando el proceso de carga de manera segura.

Permite obtener la ubicación geográfica del bastón en tiempo real mediante señales satelitales. Su información es enviada a la página web para facilitar el seguimiento y monitoreo de la ubicación del bastón.

Actúa como una interfaz de potencia entre el ESP32 y los motores vibradores. Permite controlar las vibraciones del bastón de forma segura, entregando la corriente necesaria sin sobrecargar el microcontrolador.

¿Dónde está el bastón?


Vision Stick puede compartir su ubicación utilizando el módulo GPS NEO-8M conectado al ESP32. Para enviar los datos, el ESP32 se conecta a la zona WiFi del celular de la persona, permitiendo que los familiares puedan consultar la última ubicación registrada del bastón.
