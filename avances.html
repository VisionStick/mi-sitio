Avances semanales

Registro ordenado del desarrollo de Vision Stick. Usa el buscador para encontrar una semana específica y haz clic para ver el detalle general.

Semana 1

Planificación inicial

Definición del problema, organización del equipo, asignación de tareas y primeras ideas para el diseño del prototipo.

  • Se definió la idea central del proyecto Vision Stick.
  • Se asignaron roles de trabajo por integrante.
  • Se comenzó la búsqueda de materiales y posibles componentes.
  • Se estableció la necesidad de documentar el proceso semanalmente.
Semana 2

Investigación de componentes

Búsqueda de información sobre sensores, ESP32, formas de alerta y estructura del bastón.

  • Se investigó el funcionamiento de sensores ultrasónicos.
  • Se revisó el uso del ESP32 como controlador del sistema.
  • Se definió el uso de motores vibradores como forma de alerta.
  • Se compararon opciones para alimentar el prototipo con batería recargable.
Semana 3

Diseño de página y prototipo

Construcción de la primera versión de la página web y organización visual del proyecto.

  • Se creó una página inicial para presentar el proyecto.
  • Se organizaron secciones como inicio, avances y equipo.
  • Se comenzó a definir la estética visual de Vision Stick.
  • Se preparó contenido para explicar el propósito del bastón.
Semana 4

Primer prototipo y presentación

Construcción del primer prototipo funcional y preparación del material para presentar el proyecto.

  • Se realizaron conexiones iniciales entre sensores y ESP32.
  • Se planteó la distancia e tener en cuenta para el proyecto
  • Se preparó la presentación del proyecto para la clase, profesor y ayudante.
  • Se probaron componentes y alimentación del sistema.
Semana 5

Planificación y mejora del diseño

Semana enfocada en planificación del proyecto, organización del equipo e investigación de nuevas funcionalidades para Vision Stick.

  • Se organizó la planificación de las siguientes etapas del proyecto.
  • Se diseñó un modelo preliminar del bastón con distribución estimada de componentes.
  • Se analizaron posibles ubicaciones internas para sensores, batería y motores vibradores.
  • Se investigó la integración de GPS para localización del bastón.
